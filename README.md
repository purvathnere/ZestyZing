    <h1>ZestyZing - Project Overview</h1>
    <p>ZestyZing is a comprehensive system designed for managing customers, inventory, and orders. It offers secure authentication, 
    role-based access control, and database integration using MySQL. Built with Spring Boot and Thymeleaf, the application provides 
    a seamless experience for admin and staff members.</p>

    <h2>Features</h2>
    <div class="features">
        <ul class="list-style">
            <li><strong>Customer Management</strong>: Easily add, update, and delete customer information.</li>
            <li><strong>Inventory Management</strong>: Keep track of your inventory items, including stock levels and pricing.</li>
            <li><strong>Order Management</strong>: Manage customer orders, including order creation, updates, and status tracking.</li>
            <li><strong>User Authentication</strong>: Secure login and authentication for admin and staff members.</li>
            <li><strong>Role-Based Access Control</strong>: Define roles and permissions for different user types.</li>
            <li><strong>Thymeleaf Templates</strong>: Utilizes Thymeleaf for dynamic HTML templates.</li>
            <li><strong>Database Integration</strong>: Integrated with MySQL for data storage and retrieval.</li>
        </ul>
    </div>

    <h2>Technology Stack</h2>
    <div class="tech-stack">
        <ul>
            <li><strong>Backend</strong>: Spring Boot, Java 8, Spring MVC, Spring Data JPA (Hibernate)</li>
            <li><strong>Frontend</strong>: Thymeleaf, HTML, CSS, JavaScript</li>
            <li><strong>Database</strong>: MySQL</li>
            <li><strong>IDE</strong>: Eclipse, Spring Tool Suite (STS)</li>
        </ul>
    </div>

    <h2>Prerequisites</h2>
